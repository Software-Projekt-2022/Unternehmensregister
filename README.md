### Building the API

### Running the API
```bash
java -jar .\build\libs\RestApi-X.X.jar --CJ_HOST=URL --CJ_PORT=PORT --CJ_DB=DBNAME --CJ_USER=DBUSER --CJ_PW=DBPASSWORD
```
